# Kissi Kingdom Civil Registry

Sovereign citizen enrollment, identification, and document issuance platform for the Kingdom of the Kissi people.

## Features

- **Citizen Registration** — Enroll citizens with unique KK-CRN identification numbers
- **Photo Upload** — Passport-quality photo with crop, zoom, and reposition
- **Birth Certificates** — Sovereign birth registration certificates
- **National ID Cards** — Official Kissi Kingdom identification with photo
- **Sovereign Passport** — ICAO 9303 compliant travel document with MRZ zone
- **Dual Citizenship** — Kissi Kingdom / United States dual citizenship certificates

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- localStorage for data persistence

## Getting Started

```bash
bun install
bun dev
```
